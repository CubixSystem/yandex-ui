### Кастомная тема

Вы также можете настроить внешний вид Progress с помощью токенов:

`progress-outer-fill-color` - цвет подложки
`progress-inner-fill-color:` - цвет внутренней заливки

```yaml
progress:
  outer:
    fillColor:
      value: "#ebedf1"
  inner:
    fillColor:
      value: "#6c99fd"
```
