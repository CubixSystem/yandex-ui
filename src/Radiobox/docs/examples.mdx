### Вид радиопереключателя

Для различного оформления радиопереключателей установите свойство `view` в значение `"default"` или `"outline"`.

### Классический дизайн

Чтобы использовать классический дизайн для радиопереключателей, установите свойство `theme` в одно из следующих значений: `"normal"`, `"pseudo"`.

<Example
    component={RadioboxTheme}
    source={themeRadioboxSource}
/>

### Размер радиопереключателя

Чтобы изменить размер радиопереключателя, установите свойство `size` в одно из следующих значений: `"m"`, `"s"`.
